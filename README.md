# 明日方舟电子通行认证 PCB

一款类似手机游戏《明日方舟》中的通行认证的电子挂牌。

当前版本：Ver0.6

本项目存放通行认证的PCB电路设计。PCB为群友“你说得对”使用嘉立创EDA进行Layout。

USB布线比我的kicad好，而且只需要二层板。如果你需要查看原始kicad设计文件，请在左上角将分支切换为：onlykicad。

设计。工艺参数参考嘉立创二层板，设计厚度为1mm。
